**Step 1 — Sync a node to create it (replace YOUR_PROJECT_ID with a real project id from your DB):**
```bash
curl -s -X POST http://localhost:5000/api/v1/workflow/YOUR_PROJECT_ID/sync \
  -H 'Content-Type: application/json' \
  -d '{"nodes":[{"nodeId":"test-node-1","nodeType":"text"}]}'
```

**Step 2 — Update node metadata:**
```bash
curl -s -X PATCH http://localhost:5000/api/v1/node/test-node-1 \
  -H 'Content-Type: application/json' \
  -d '{"requirement":"Build login page","prompt":"Create a Vue 3 login component"}'
```
Expected: JSON entity with `requirement: "Build login page"`, `status: "pending"` (unchanged)

**Step 3 — Update status (creates history snapshot):**
```bash
curl -s -X PATCH http://localhost:5000/api/v1/node/test-node-1/status \
  -H 'Content-Type: application/json' \
  -d '{"status":"completed"}'
```
Expected: JSON entity with `status: "completed"`

**Step 4 — Record AI execution result:**
```bash
curl -s -X POST http://localhost:5000/api/v1/node/test-node-1/history \
  -H 'Content-Type: application/json' \
  -d '{"result":"Login component created successfully","executedBy":"claude-code"}'
```
Expected: History entity with `createdBy: "claude-code"`

**Step 5 — Get history (reverse-chrono):**
```bash
curl -s http://localhost:5000/api/v1/node/test-node-1/history
```
Expected: JSON array with at least 1 entry (from Step 3 snapshot and Step 4), ordered newest first.

**Step 6 — Verify 404 for unknown node:**
```bash
curl -s -o /dev/null -w "%{http_code}" http://localhost:5000/api/v1/node/nonexistent-node/history
```
Expected: `404`
