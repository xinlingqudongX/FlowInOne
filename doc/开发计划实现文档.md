## 一、项目概述

### 1.1 项目定位
FlowInOne 是一个工作流程图编辑器，用于将业务功能拆解为有依赖关系的节点图谱（DAG/知识图谱），并导出为 AI 可消费的结构化数据。

### 1.2 核心目标
- 将项目的功能与依赖关系抽象为工作流图谱
- 前端基于 File System API 加载本地目录作为工作区，将流程文档写回本地
- 后端负责数据记录与 AI 能力提供

### 1.3 用户角色
- 无需登录即可使用
- 主要面向开发人员、产品经理、业务架构师
